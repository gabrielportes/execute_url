# Shell Script para rodar uma URL em várias licenças (VERSÃO BETA)

- Para utilizá-lo, basta baixá-lo para alguma pasta de seu PC
- Torná-lo executavél: chmod +x execute_url.sh
- Executá-lo: ./execute_url.sh
- Informar o path completo do arquivo que contém as licenças
- Informar a URL a ser executada (utilizar a variável '$licenca', essa palavra reservada será substituída pelas licenças do arquivo informado anteriormente)
